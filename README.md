# Wine-Classification-Using-Neural-Networks-with-TensorFlow

# Project Statement

* To develop a neural network model using TensorFlow to classify wine samples into different categories based on their chemical properties with good accuracy.

# Objective

* To understand the working of Neural Networks
* To implement a classification model using TensorFlow

* To preprocess data using scaling and encoding

* To evaluate model performance using accuracy, confusion matrix, and classification report

# Dataset Description

* Dataset Name: Wine Dataset

* Source: sklearn.datasets.load_wine

* Total Samples: 178

* Number of Features: 13

* Number of Classes: 3 (class_0, class_1, class_2)

#  Features Include:

*  Alcohol, Malic Acid, Ash, Magnesium, Total Phenols, Flavanoids, Color Intensity, Hue, etc.

#  Model Used

* Artificial Neural Network (ANN)

* ReLU activation (hidden layers)

* Softmax activation (output layer)

# Evaluation Metrics

* Accuracy

* Confusion Matrix

* Classification Report

# Results

* The model achieved high accuracy on the test dataset.

* Most predictions are correctly classified, as shown by the confusion matrix.

* The classification report shows good precision, recall, and F1-score for all classes.

* The model generalizes well to unseen data.

# Visualization Explanation

* Accuracy Curve: Shows how training and validation accuracy improve over epochs.

* Loss Curve: Shows how training and validation loss decrease, indicating effective learning.

* Confusion Matrix: Visualizes correct and incorrect predictions for each wine class.

# Conclusion

* The neural network successfully classifies wine samples with good accuracy. This project demonstrates how deep learning can be applied to structured datasets using TensorFlow.
